# FAIRmapper

Create API adapters without Java code using YAML + JSLT transforms.

## Quick Start

```bash
./fairmapper --help
./fairmapper validate beacon-v2
./fairmapper test beacon-v2 -v
./fairmapper e2e beacon-v2 --server http://localhost:8080 --schema mySchema
```

Auto-builds on first run. See [.plan/plans/fairmapper.md](../.plan/plans/fairmapper.md) for details.
